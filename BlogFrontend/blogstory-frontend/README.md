# BlogStory

**BlogStory** is a high-performance backend **API** with a **React-based** interface, built using **FastAPI** . It provides secure user authentication, comprehensive blog management, and token-based access via **JWT**, all structured for clean, modular, and easily extensible code.

---

## Live Demo

> **API is deployed using [Render](https://render.com/)**  
> Visit the live API: [https://blog-story.onrender.com/docs](https://blog-story.onrender.com/docs)

---

## Features

- User Registration & Login
- JWT Authentication for secure API access
- Create, Read, Update, Delete (CRUD) for blogs
- Link blogs to their respective creators (users)
- Fetch blogs by ID or all
- Modular route handling using FastAPI routers
- Pydantic-based request/response models
- SQLAlchemy ORM for DB interaction

---

## Tech Stack

- Python
- ReactJS
- **Framework:** FastAPI
- **Database:** SQLite
- **ORM:** SQLAlchemy
- **Authentication:** JWT (JSON Web Tokens)
- **Validation:** Pydantic
- **Server:** Uvicorn
- **Deployment:** Render
